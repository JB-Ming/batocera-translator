# 此資料夾用於存放應用程式設定和使用者資料
# 請勿上傳 settings.json（包含 API Key 等敏感資料）

## 資料夾結構
- `settings.json` - 應用程式設定（自動建立）
- `settings.example.json` - 設定檔範例
- `backups/` - gamelist.xml 備份
- `cache/` - 快取資料
- `dictionaries/` - 使用者自訂字典

## 首次使用
1. 程式會自動建立 `settings.json`
2. 如需手動設定，可複製 `settings.example.json` 為 `settings.json` 並修改
