# Batocera Gamelist 翻譯工具 - 語系包貢獻指南

感謝您有興趣貢獻語系包！這份指南將幫助您瞭解如何提交翻譯。

---

## 語系包格式

語系包是 JSON 檔案，每個平台一個檔案，放置於 `language_packs/{語系代碼}/` 目錄下。

### 檔案結構

```
language_packs/
├── zh-TW/          # 繁體中文
│   ├── nes.json
│   ├── snes.json
│   └── ...
├── zh-CN/          # 簡體中文
│   └── ...
├── ja/             # 日文
│   └── ...
└── ko/             # 韓文
    └── ...
```

### JSON 格式

```json
{
  "遊戲名稱 (年份)(發行商)": {
    "key": "遊戲名稱 (年份)(發行商)",
    "original_name": "遊戲名稱 (年份)(發行商)",
    "name": "翻譯名稱",
    "name_source": "pack",
    "original_desc": "原始英文描述",
    "desc": "翻譯後的描述",
    "desc_source": "pack",
    "needs_retranslate": false,
    "name_translated_at": "",
    "desc_translated_at": "",
    "original_name_hash": "",
    "original_desc_hash": ""
  }
}
```

### 欄位說明

| 欄位 | 必填 | 說明 |
|------|------|------|
| `key` | ✅ | 遊戲識別 Key，與 JSON 物件的 key 相同 |
| `original_name` | ✅ | 原始遊戲名稱 |
| `name` | ✅ | 翻譯後的名稱（空字串表示未翻譯） |
| `name_source` | ✅ | 來源：`pack`（語系包）、`api`（API翻譯）、`keep`（保留原文）、空字串（未翻譯） |
| `original_desc` | ❌ | 原始英文描述 |
| `desc` | ❌ | 翻譯後的描述 |
| `desc_source` | ❌ | 來源：`pack`（語系包）、`api`（API翻譯）、空字串（未翻譯） |
| `needs_retranslate` | ❌ | 是否需要重新翻譯 |
| `name_translated_at` | ❌ | 名稱翻譯時間 |
| `desc_translated_at` | ❌ | 描述翻譯時間 |
| `original_name_hash` | ❌ | 原始名稱 Hash（用於偵測變更） |
| `original_desc_hash` | ❌ | 原始描述 Hash（用於偵測變更） |

## 如何貢獻

### 方式一：直接編輯

1. Fork 本專案
2. 在 `language_packs/{語系}/` 下新增或編輯 JSON 檔案
3. 提交 Pull Request

### 方式二：使用工具匯出

1. 使用本工具翻譯遊戲
2. 點選「匯出字典」
3. 將匯出的 JSON 檔案提交 Pull Request

---

## 翻譯指南

### 遊戲名稱

- **優先使用官方中文名稱**：如遊戲有官方中文版，使用官方譯名
- **次選維基百科譯名**：有中文維基百科頁面的遊戲，使用維基百科標題
- **再次選巴哈姆特譯名**：台灣常用的遊戲譯名
- **保留原文**：部分遊戲約定俗成使用英文名（如 F-Zero、NBA 2K）

### 遊戲描述

- 描述可從維基百科或官方網站翻譯
- 長度建議 100-300 字
- 避免劇透主要劇情

---

## 品質要求

- ✅ 使用正確的標點符號（繁中用全形，簡中用全形）
- ✅ 人名、地名保持一致性
- ✅ 避免機器翻譯的生硬用語
- ❌ 不要包含廣告或無關內容

---

## 審核流程

1. 提交 PR 後，維護者會審核翻譯品質
2. 需修改時會在 PR 中留言
3. 審核通過後合併至主線
4. 下一版程式會包含更新的語系包
